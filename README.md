# trywright — issues & feedback

Public issue tracker for [Trywright](https://trywright.dev) — a local 
control panel for Playwright.

**Install:** `npm install -g trywright`  
**Website:** [trywright.dev](https://trywright.dev)  
**Docs:** [trywright.dev/docs](https://trywright.dev/docs)

---

Found a bug or have a suggestion? Open an issue below.  
Please include your Trywright version (`trywright --version`),  
OS, Node.js version, and the browser you were using.
