# trywright — issues & feedback

Public issue tracker for [Trywright](https://trywright.dev).

**→ [Report a bug](https://github.com/abhivaikar/trywright-issues/issues/new?template=bug_report.yml)**  
**→ [Request a feature](https://github.com/abhivaikar/trywright-issues/issues/new?template=feature_request.yml)**

---

### Before filing an issue

- Check the [docs](https://trywright.dev/docs) — your question may already be answered
- Search [existing issues](https://github.com/abhivaikar/trywright-issues/issues) to avoid duplicates
- Include your Trywright version (`trywright --version`), OS, and Node.js version

---

**Install:** `npm install -g trywright`  
**Website:** [trywright.dev](https://trywright.dev)  
**Docs:** [trywright.dev/docs](https://trywright.dev/docs)
